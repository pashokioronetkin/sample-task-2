# Система управления экзаменами

## Описание проекта
Программа для демонстрации экзаменационного процесса с использованием:
- Очереди с приоритетами (по рейтингу студентов)
- Ad hoc полиморфизма
- Паттерна Abstract Factory
- Файлового хранения данных

## Реализованные задания

### 1. Статический (Ad hoc) полиморфизм
Демонстрация через интерфейс `Calculator` с разными реализациями:
- `IntCalculator` - для целых чисел
- `FloatCalculator` - для чисел с плавающей точкой

### 2. Классы Студент и Экзамен
- **Student**: id, name, rating, mark
- **Exam**: очередь с приоритетами, логика проведения экзамена
- **PriorityQueue**: реализация с использованием container/heap

### 3. Демонстрация экзамена
- Чтение данных из текстового файла
- Сортировка студентов по рейтингу (выше рейтинг → раньше сдает)
- При равном рейтинге используется FIFO
- Автоматический расчет оценок

### 4. Abstract Factory
Паттерн для создания репозиториев:
- `RepositoryFactory` - абстрактная фабрика
- `InMemoryRepositoryFactory` - конкретная реализация


## 5. Требования
- Go 1.21 или выше

## Установка и запуск

```bash
# 1. Создаем структуру проекта
mkdir -p sample-task/{cmd,internal/domain,internal/repository,internal/service,internal/factory,internal/polymorphism,pkg/diagrams,data}
cd sample-task

# 2. Инициализация модуля
go mod init go/sample-task

# 3. Создаем файлы (скопировать код выше)

# 4. Создаем файл с данными
cat > data/students.txt << 'EOF'
# Формат: ID,Имя,Рейтинг,Оценка
1,Иванов Иван,85.5,0
2,Петров Петр,92.0,0
3,Сидорова Анна,78.3,0
4,Козлов Дмитрий,92.0,0
5,Михайлова Елена,88.7,0
EOF

# 5. Запуск программы
go run cmd/main.go


Пример данных (data/students.txt)

Формат файла студентов
ID,Имя,Рейтинг,Оценка
ID: уникальный номер (целое число)

Имя: ФИО студента

Рейтинг: число от 0 до 100

Оценка: 0 (еще не сдал) или от 1 до 10

---
Бизнес-правила
Студенты сортируются по убыванию рейтинга
При равном рейтинге используется FIFO (кто раньше добавлен)

Оценка рассчитывается автоматически:
90-100: 10
80-89: 9
70-79: 8
60-69: 7
50-59: 6
ниже 50: 5
